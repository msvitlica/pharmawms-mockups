# PharmaWMS — UI mockups

Statički UI mockup-i za veleprodaju lijekova (GDP/GxP) — svijetla i tamna tema.

Otvori `index.html`. Parametri za pregled varijanti:
`?theme=dark` i `?accent=teal|indigo|blue|emerald`.

Čista statika (HTML + Tailwind CDN). Tema/akcenat: `tokens.css`, `accents.css`.
Interakcije (tema, tabovi, collapse, prekidači): `app.js`.
